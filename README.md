# Inventory Management System

A Full Stack Inventory Management System developed using React, FastAPI, SQLAlchemy, and MySQL.

## Features

- Add Product
- Update Product
- Delete Product
- Search Product
- Dynamic Product Table
- MySQL Database Integration
- SQLAlchemy ORM
- FastAPI REST APIs
- Responsive React UI

## Tech Stack

### Frontend
- React
- Axios
- CSS

### Backend
- FastAPI
- SQLAlchemy
- MySQL
- PyMySQL

## Database

MySQL
